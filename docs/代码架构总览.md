# selector 目录 Python 文件总览

只列工作目录顶层 + `2B/` + `2B/data/v2/`，**不包含** `verl-main/` 第三方依赖和 `old/` 历史脚本。

## 主管线（端到端选块 + 评测）

- [longbench_e2e.py](../longbench_e2e.py) — 主 pipeline v1：reranker top-p 池 → 0.8B 贪婪 vs reranker top-k → 35B 读 → 写 jsonl。带 SHRINK 选项
- [2B/data/v2/longbench_e2e_v2.py](data/v2/longbench_e2e_v2.py) — v2：纯 0.8B solo（empty baseline + softmax top-p + 贪婪）vs reranker top-k，0.8B 不依赖 reranker
- [longbench_e2e_v2.py](../longbench_e2e_v2.py) — 桩文件，仅指向 `2B/data/v2/` 那个
- [longbench_shrink.py](../longbench_shrink.py) — 选好块再去掉最弱的 N 个的 ablation
- [longbench_expand.py](../longbench_expand.py) — 把 both_wrong 的行加 N 个补块再读
- [e2e_qwen_vs_reranker.py](../e2e_qwen_vs_reranker.py) — 旧版端到端对比（被 longbench_e2e 取代）
